Hello, I'm Mika. I'm a software & web developer currently living in Leipzig, Germany. I have professional experience in Java & Visual Basic .NET, with a focus on applications that interface with large databases via SQL. I also have experience in professionally developing websites with HTML, CSS and Javascript. In my free time I develop with Python, Ruby on Rails, Haskell and Processing - as well as electronics prototyping with the Arduino platform.

My academic interests in computing and software are mainly focused on nonlinear dynamics, distributed and software defined networking, information theory, complexity and emergent behaviours, autonomous agents and generative art. I also maintain an interest in both the technological and social aspects of security, encryption, anonymity and privacy and their implementation in web based communications. Alongside these, I am deeply interested in social and historical topics within computing, such as as the development of encryption/cryptography, the origins of personal computing and the role of technology in labour dynamics.

Multi/Timer

A single page app built out of frustration with the lack of functionality and bad aesthetics of other online timers. Allows multiple timers at once, for more complex cooking tasks. Each timer is customisable so that differentation between them is easy. Hosted at multitimer.io and the code is available on Github.

HTML5/CSS3, Javascript, jQuery

Mainframe

A web game based on bad cyberpunk movies and the Warioware games. Built using the desktop and mobile HTML5 game framework Phaser. It can be played on itch.io and the code is available on Github.

Mainframe was featured on RockPaperShotgun's best free games of the week and Geek Squad's top free games of March 2016.

Javascript, Phaser, Game Design

Museumzeit

A website built with Python, using the Flask microframework. The aim of Museumzeit is to display, at a glance, the opening hours of museums/art galleries of interest in a given city. Still under development. Hosted on Heroku and the code is available on Github.

Python, Flask, SQL, HTML5/CSS3, jQuery

Avari

A custom minimalist 2 column mobile-friendly wordpress theme, with emerald green accents and an owl. Emphasis on using spacing and typography to present large bodies of text. Degrades gracefully on mobile resolutions so that the right hand column becomes a collapsed accordian menu at the bottom of the page. Developed for Notes & Commentaries and the theme is available on Github.

HTML5/CSS3, Wordpress Theme Development, Jquery, PHP
